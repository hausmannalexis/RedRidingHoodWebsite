Global Attributes
These can be used on most HTML elements:

    accesskey: Specifies a shortcut key to activate/focus an element1
    .
    class: Assigns one or more CSS class names to an element1
    .
    contenteditable: Indicates whether the element's content is editable1
    .
    data-*: Used to store custom data private to the page or application1
    .
    dir: Specifies the text direction for the content in an element1
    .
    draggable: Specifies whether an element is draggable1
    .
    hidden: Hides an element from view1
    .
    id: Provides a unique identifier for an element1
    .
    lang: Specifies the language of the element's content1
    .
    spellcheck: Indicates whether the element should have its spelling checked1
    .
    style: Specifies inline CSS styles for an element1
    .
    tabindex: Defines the tabbing order of an element1
    .
    title: Provides additional information about an element, often shown as a tooltip1
    .
